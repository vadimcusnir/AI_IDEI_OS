# Idea Extraction Pipeline

## Overview

The extraction pipeline transforms raw content into structured knowledge units (neurons) through a multi-stage AI process.

## Pipeline Stages

```
1. Upload → Episode created
2. Transcription → Text extracted
3. Semantic Chunking → 200-800 token segments
4. AI Analysis → 3-axis extraction (9 specialized prompts)
5. Neuron Generation → Atomic knowledge units created
6. Entity Projection → Neurons → Knowledge Graph entities
7. IdeaRank → Graph-based scoring
```

## Extraction Axes

### Axis 1: Internal/Psychological
- Cognitive patterns and mental models
- Decision frameworks and heuristics
- Behavioral insights and biases

### Axis 2: Narrative
- Storytelling structures
- Persuasion techniques
- Argument maps and rhetorical formulas

### Axis 3: Commercial/JTBD
- Market insights and value propositions
- Growth strategies and pricing models
- Jobs-to-be-done patterns

## Neuron Categories

| Category | Description |
|----------|-------------|
| transcript | Raw content reference |
| insight | Extracted knowledge pattern |
| framework | Structured thinking model |
| strategy | Strategic/commercial approach |
| formula | Actionable copywriting/persuasion formula |
| pattern | Recurring structural element |
| avatar | Person/audience profile |
| argument_map | Logical argument structure |
| narrative | Story/narrative pattern |
| psychological | Cognitive/behavioral insight |
| commercial | Business/market insight |

## Neuron Lifecycle

```
ingested → structured → active → capitalized → compounded
```

Each stage represents increasing refinement and economic value of the knowledge unit.

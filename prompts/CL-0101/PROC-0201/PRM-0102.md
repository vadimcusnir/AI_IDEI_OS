## Prompt Body

Reconstruiește information architecture-ul întregului site pornind din modelul de UI graph: Home, Extractor, Neurons, Services, Jobs, Library, Credits și ecranele contextuale asociate. Impune reguli: zero dead-ends, minimum 2 ieșiri utile per ecran important, depth max 3, pipeline persistent, context propagation. Livrează: sitemap canonic, route map, relații între noduri, pagini de eliminat, pagini de comasat, nomenclatură finală a meniurilor.
